# 📝 Simple Blog Application (MERN Stack)

## 📌 Project Overview
This is a full-stack blog application built using the MERN stack. It allows users to create, view, and manage blog posts through a clean and responsive interface. The application follows a client-server architecture and uses RESTful APIs for communication between frontend and backend.

---

## 🚀 Features
- ✍️ Create new blog posts  
- 📖 View all blogs  
- 🔍 Read individual blog posts  
- ✏️ Update blog posts (optional)  
- ❌ Delete blog posts (optional)  
- 🔗 REST API integration  
- ⚡ Fast and responsive UI
